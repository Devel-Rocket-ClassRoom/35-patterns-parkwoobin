# 성적 평가기

학생의 점수를 입력받아 학점과 상태를 출력하는 성적 평가 프로그램을 작성하세요.

**요구사항**

1. `GetGrade` 메서드: 점수를 받아 학점을 반환
   - 90점 이상: "A"
   - 80점 이상 90점 미만: "B"
   - 70점 이상 80점 미만: "C"
   - 60점 이상 70점 미만: "D"
   - 60점 미만: "F"

2. `GetStatus` 메서드: 점수를 받아 상태를 반환
   - 95점 이상: "최우수"
   - 90점 이상 95점 미만: "우수"
   - 70점 이상 90점 미만: "보통"
   - 40점 이상 70점 미만: "노력 필요"
   - 40점 미만: "재수강 권장"

3. `IsPassingGrade` 메서드: 점수를 받아 합격 여부를 반환
   - 60점 이상이면 합격 (true)

**구현 조건**

- 모든 메서드는 switch 식과 관계 패턴을 사용하여 구현
- `IsPassingGrade`는 패턴 결합자 `and`를 활용

## 예상 실행 결과

```
=== 성적 평가기 ===
95점: A (최우수) - 합격
87점: B (보통) - 합격
73점: C (보통) - 합격
65점: D (노력 필요) - 합격
45점: F (노력 필요) - 불합격
30점: F (재수강 권장) - 불합격
```
