# 도형 분류기

다양한 도형 객체를 분류하고 넓이를 계산하는 프로그램을 작성하세요.

**요구사항**

1. `Shape` 기본 클래스와 파생 클래스 구현:
   - `Circle`: 반지름(`Radius`) 속성
   - `Rectangle`: 너비(`Width`)와 높이(`Height`) 속성
   - `Square`: 한 변의 길이(`Side`) 속성

2. `CalculateArea` 메서드: 도형 객체를 받아 넓이를 반환
   - 원: π x 반지름²
   - 직사각형: 너비 x 높이
   - 정사각형: 한 변² (Rectangle의 특수 케이스로도 처리 가능)

3. `ClassifyShape` 메서드: 도형 객체를 받아 설명 문자열을 반환
   - 타입과 속성 패턴을 함께 사용
   - 예: 반지름이 10 이상인 원은 "큰 원", 미만은 "작은 원"
   - 예: 너비와 높이가 같은 Rectangle은 "정사각형 모양의 직사각형"

**구현 조건**

- 타입 패턴을 사용하여 도형 종류 구분
- 속성 패턴을 사용하여 크기 기반 분류
- switch 식으로 간결하게 구현

## 예상 실행 결과

```
=== 도형 분류기 ===
원(반지름: 5): 작은 원, 넓이: 78.54
원(반지름: 15): 큰 원, 넓이: 706.86
직사각형(4x6): 일반 직사각형, 넓이: 24.00
직사각형(5x5): 정사각형 모양의 직사각형, 넓이: 25.00
정사각형(7): 정사각형, 넓이: 49.00
```
